# Инструкция по размещению сайта

## Вариант 1: Netlify Drop (самый простой - без регистрации)

1. Откройте сайт: https://app.netlify.com/drop
2. Перетащите всю папку проекта (index.html, styles.css, script.js, logo.png) в окно браузера
3. Через несколько секунд ваш сайт будет доступен по ссылке типа: `https://random-name-123.netlify.app`
4. Вы можете изменить название сайта в настройках проекта

## Вариант 2: GitHub Pages (если у вас есть GitHub)

### Шаг 1: Создайте репозиторий на GitHub
1. Зайдите на https://github.com
2. Нажмите "New repository"
3. Назовите репозиторий (например, `my-website`)
4. Нажмите "Create repository"

### Шаг 2: Загрузите файлы
1. На странице репозитория нажмите "uploading an existing file"
2. Перетащите все файлы проекта: index.html, styles.css, script.js, logo.png
3. Нажмите "Commit changes"

### Шаг 3: Включите GitHub Pages
1. Зайдите в Settings → Pages
2. В разделе "Source" выберите "main" branch
3. Нажмите "Save"
4. Ваш сайт будет доступен по адресу: `https://ваш-логин.github.io/название-репозитория`

## Вариант 3: Vercel (очень просто)

1. Зайдите на https://vercel.com
2. Зарегистрируйтесь (можно через GitHub)
3. Нажмите "Add New" → "Project"
4. Перетащите папку проекта или подключите GitHub репозиторий
5. Нажмите "Deploy"
6. Сайт будет доступен по ссылке типа: `https://ваш-сайт.vercel.app`

## Вариант 4: Cloudflare Pages (бесплатно)

1. Зайдите на https://pages.cloudflare.com
2. Зарегистрируйтесь
3. Нажмите "Create a project"
4. Загрузите папку проекта
5. Нажмите "Deploy site"
6. Сайт будет доступен по ссылке

## Вариант 5: Обычный хостинг (платный)

Если вы хотите использовать платный хостинг:
1. Купите хостинг (например, Timeweb, Beget, REG.RU)
2. Загрузите файлы через FTP или файловый менеджер
3. Разместите файлы в папке `public_html` или `www`

## Рекомендация

**Для быстрого результата используйте Netlify Drop** - самый простой способ без регистрации и настройки.

**Для постоянного использования рекомендую Vercel** - очень простая настройка, бесплатно, и можно легко обновлять сайт.
